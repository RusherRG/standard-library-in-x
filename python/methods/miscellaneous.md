# GCD

```python
import math

print(math.gcd(2,4))
```
**Output** :
```python
2
```

# Fractions
```python
from fractions import Fraction

```

```python

3/2
```

# Power
```python
import math

print(math.pow(16, -10))
print(math.pow(3, 2))
```
**Output** :
```python
9.094947017729282e-13
9.0
```
